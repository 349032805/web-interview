# Web Interview

以下为一些公司的面试题，如对您有帮助，可以 Star

面试跟相亲一样，看对眼和运气都很重要

当然实力最重要，JS 的功底和 ES6 的熟练 和 jQuery 源码的熟悉都会对你的面试起到很大的帮助，
也会很看重 HTML/CSS 的布局，包括已经几乎每个公司都会问的 左右布局，Flex 的用法，移动端的自适应等等

还有一点是，几乎很多公司都会要求 Node.js

Good Luck
